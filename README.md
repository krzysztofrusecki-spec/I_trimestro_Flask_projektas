# I_trimestro_Flask_projektas

# The project ia about a mobile game "Clah Royale"

This Flask application displays pages about the game, including:
- Home page
- About the game
- Gameplay
- History
- Tips

It also contains an Easter egg

# Technologies used
- Python
- Flask
- HTML
- CSS
- JavaScript

# How to run the project
1. Install Python
2. Install Flask (pip install flask)
3. Go to the project folder and run:
   python app.py
4. Open the link that it shows (http://127.0.0.1:5000)
5. Enjoy
